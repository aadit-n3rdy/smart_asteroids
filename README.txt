SMART ASTEROIDS

-> A modern twist on the classic Asteroids
-> A rocket is controlled by the player, who's objective is to dodge incoming asteroids and stay alive for as long as possible

Objectives
-> To create a fun game which appeals to all age groups

Modules used
->pygame
->numpy
->pygame-gui
->pygame-ce (stability and cross-compatability)

Software used
->emacs (code and text editing)
->Inkscape (Sprite creation)
